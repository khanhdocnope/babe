# 🚀 Hướng Dẫn Sử Dụng Babe URL Shortener

Chào mừng bạn đến với hệ thống rút gọn liên kết tĩnh! Dưới đây là các bước đơn giản để bạn tạo và quản lý các liên kết của mình.

## 📝 Cách rút gọn một liên kết mới

1.  **Truy cập trang web:** Mở địa chỉ GitHub Pages của bạn trên trình duyệt.
2.  **Nhập URL:** Dán liên kết dài mà bạn muốn rút gọn vào ô "Nhập URL dài của bạn".
3.  **Tùy chỉnh (Không bắt buộc):**
    *   Nhấn vào **⚙️ Tùy Chọn Nâng Cao**.
    *   Nhập tên gợi nhớ vào ô **Bí Danh Tùy Chỉnh** (ví dụ: `facebook-cua-toi`, `khuyen-mai-thang-10`).
4.  **Gửi yêu cầu:** Nhấn nút **Gửi Yêu Cầu Rút Gọn**.
5.  **Xác nhận trên GitHub:**
    *   Hệ thống sẽ chuyển bạn đến trang tạo Issue trên GitHub của chính bạn.
    *   Mọi thông tin đã được điền sẵn, bạn chỉ cần nhấn nút **Submit new issue**.

## ⏳ Đợi hệ thống xử lý

*   Sau khi gửi Issue, GitHub Actions sẽ tự động chạy trong khoảng **30 - 60 giây**.
*   Khi thấy Issue được tự động chuyển sang trạng thái **Closed** (Đã đóng), nghĩa là link của bạn đã sẵn sàng!

## 🔗 Cách lấy và chia sẻ liên kết

1.  Quay lại trang chủ của ứng dụng.
2.  Nhấn nút **🔄 Làm Mới** ở phần danh sách liên kết.
3.  Tại đây bạn có thể:
    *   **Click vào mã rút gọn:** Để mở trực tiếp liên kết.
    *   **Nhấn biểu tượng 📋:** Để sao chép nhanh URL ngắn vào bộ nhớ đệm.
    *   **Nhấn biểu tượng 📱:** Để hiển thị **Mã QR**. Bạn có thể chụp ảnh màn hình mã này để in ấn hoặc gửi cho bạn bè quét bằng điện thoại.

## 💡 Lưu ý quan trọng
*   Nếu link chưa xuất hiện ngay, hãy đợi thêm một chút vì GitHub Pages cần thời gian ngắn để cập nhật các file tĩnh mới.
*   Chỉ sử dụng chữ cái, số và dấu gạch ngang cho bí danh tùy chỉnh.
